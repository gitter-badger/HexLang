HEX LANG

a simple compiled programming language

Install Requirements on Windows

Download Python 3.x for Windows from here. Execute the downloaded file and install Python. This should also give you Pip on your system. Download Git for Windows from here. Execute the downloaded file. Then, download Clang from here. Check that all of these are available from the command line. Then, open a terminal and run this command:

$ python -m pip install git+https://github.com/RealAAbraham/HexLang.git

Install Requirements on Mac OSX

First, we need to install Homebrew. Run this command:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

Then, we need install Clang, Git and Python. Run this command:

brew install git clang python3

Check that all four above-mentioned programs are available from the command-line. Then, run this command:

$ python3 -m pip install git+https://github.com/RealAAbraham/HexLang.git

Install Requirements on Linux

We can install all requirements like this:

sudo apt-get update && sudo apt-get upgrade -y && sudo apt-get install -y clang python3 git python3-pip

Check that all four above-mentioned programs are available from the command-line. Then, we need to install Hex itself. Run this command:

$ python3 -m pip install git+https://github.com/RealAAbraham/HexLang.git
