HEX LANG

a simple compiled programming language

Building Hex is very simple. You should have Git, Make, Python 3+, the GCC and Pip installed. After that, execute these steps in this order:

First, we need to clone the source tree. This happens with Git. Open up a terminal in a folder of your choice and execute this command.

$ git clone https://github.com/RealAAbraham/HexLang.git

Change directory and install all development dependencies executing these commands:

cd HexLang

$ python -m pip install -r requirements.txt

Finally, we build the Python package with this command:

$ python setup.py sdist

To install it and test the distribution, run this command:

$ python -m pip install dist/*
